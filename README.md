## Sviluppo

- Per editare il template importare il file `template.tpl` in Google Tag Manager Template Editor e riesportarlo una volta completata la modifica
- Nel file esportato viene rimossa la riga con le categorie, questa modifica non va committata (Ã¨ una stranezza, vedi guida alla pubblicazione)
